### What was excluded

The 30+ source files of `//common:common` (e.g. `common.cpp`, `chat.cpp`, `sampling.cpp`, etc.) were excluded because they depend on `llama.h`, `ggml.h`, `ggml-opt.h`, `llama-cpp.h` — the core llama.cpp API headers — which are in `src/` and not built by this Bazel setup.

### Biggest blocker (resolved)

`includes = ["."]` on the jinja library caused `common/jinja/string.h` to shadow the system `<string.h>` via `-I common/jinja/`, breaking `<cstring>`. The fix was to use only `includes = [".."]`, relying on GCC's standard behavior of searching the source file's own directory first for `#include "header.h"` style includes.
